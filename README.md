# 🛒 E-Commerce DBMS Project

This project is a simple E-Commerce web application built using:
- MySQL (Database)
- Python Flask (Backend)
- HTML, CSS (Frontend)

## 🚀 Features
- View products
- Search products
- Sort by price
- Clean UI

## 🧠 Database
Includes:
- Users
- Products
- Orders
- Payments

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run server:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/
